# Back Cursos

Projeto desenvolvido utilizando NodeJS</span>, para prática de desenvolvimento Back-end

## Começando

### Primeiro passo

Clone o repositório
`git clone https://github.com/Gusttavo-Larios/back-moodle.git`

### Segundo passo

Instale as dependências
`yarn install`

### Terceiro passo

Execute o projeto
`yarn stdev`
